# Layered AI–XAI Framework for Bamboo Culm Screening

This repository contains the codes and notebooks for a layered AI–XAI framework developed for multi-criteria screening of structural bamboo culms, with prediction targets including bending strength (BS), modulus of elasticity (MOE), and embodied carbon (EC).
